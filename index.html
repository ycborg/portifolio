Conhecimentos

  • C

    C é uma linguagem de programação de propósito geral, amplamente utilizada para desenvolver sistemas operacionais, compiladores, dispositivos embarcados e uma variedade de aplicativos de software.

  • HTML5

    HTML5 é a quinta e última versão do HTML, sigla de Hypertext Markup Language, uma das primeiras e principais linguagens de marcação da web.

  • CSS3

    CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets (ou simplesmente CSS), onde se define estilos para seu projeto web.

  • GitHub

    GitHub é uma plataforma de hospedagem de código-fonte que permite aos desenvolvedores colaborar em projetos de software usando o controle de versão Git.

  • VSCode

    VSCode, ou Visual Studio Code, é um editor de código-fonte desenvolvido pela Microsoft. É conhecido por sua leveza, velocidade e extensibilidade, além de suportar uma ampla variedade de linguagens de programação.
